## 概要

各地方における平年値より大きくかけ離れた気象状態が今後数週間程度続くと予想された場合に発表

**資料名** <br/>
地方気象情報（社会的に影響の大きい天候に関する情報）

**配信形式** <br/>
気象庁防災情報XMLフォーマット

**データ種類コード** <br/>
VPCJ50

**/Report/Control/Title** <br/>
地方天候情報

**WebSocket圧縮形式** <br/>
gzip
