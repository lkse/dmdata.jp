---

## 概要
海上気象観測所（観測船等）による高層気象観測データ（風向・風速、気温、高度、露点等）（TEMP-SHIP報、PILOT-SHIP報 ）。

**2025年05月ごろに配信終了予定**

**資料名** <br/>
海上気象実況報

**配信形式** <br/>
国際気象通報式第8版 FM36 TEMP SHIP－海上高層実況気象通報式 <br/>
国際気象通報式第8版 FM33 PILOT SHIP－海上高層風実況気象通報式

**データ種類コード** <br/>
U@VX01 (@=E,K,L,P,S) <br/>
UGVX20

**WebSocket圧縮形式** <br/>
なし
