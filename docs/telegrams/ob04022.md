## 概要

定置地上観測所からの高層実況値（高度、気温、気温と露点温度の差、風向・風速等）（TEMP報）。

**資料名** <br/>
地上高層実況気象報 バイナリ形式

**配信形式** <br/>
国際気象通報式第8版 FM35 TEMP－地上高層実況気象通報式

**データ種類コード** <br/>
IU#&ii (#&=KL,SC ii=01,02) <br/>
IUSCii (ii=01~12) <br/>
IU@#ii (@=K,S, #=A~L,N,S,T,X)

**WebSocket圧縮形式** <br/>
なし
