## 概要
各府県予報区における気象の実況・予測を発表

**2026年出水期を目途に平文の範疇で運用の整理あり、2028年以降に配信終了**

**資料名** <br/>
 府県気象情報
 
**配信形式** <br/>
 気象庁防災情報XMLフォーマット

**データ種類コード** <br/>
 VPFJ50

**/Report/Control/Title** <br/>
 府県気象情報
 
**WebSocket圧縮形式** <br/>
 gzip
