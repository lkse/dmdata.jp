## 概要
「噴火速報」又は「噴火に関する火山観測報（連続噴火の停止を報じる場合を除く）」を発表後、速やかに上空の風の状況をお知らせする。

**資料名** <br/>
 推定噴煙流向報
 
**配信形式** <br/>
 気象庁防災情報XMLフォーマット

**データ種類コード** <br/>
 VFVO60
 
**/Report/Control/Title** <br/>
 推定噴煙流向報

**WebSocket圧縮形式** <br/>
 gzip
