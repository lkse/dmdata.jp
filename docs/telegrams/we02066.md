## 概要

大雪に関する警報・注意報（大雪特別警報、大雪警報、大雪注意報）について府県予報区ごとに発表

**2026年出水期を目途に配信開始**

**資料名** <br/>
大雪に関する警報・注意報

**配信形式** <br/>
気象庁防災情報XMLフォーマット

**データ種類コード** <br/>
VPWW60

**/Report/Control/Title** <br/>
気象警報・注意報（Ｒ０６）（大雪）

**WebSocket圧縮形式** <br/>
gzip
