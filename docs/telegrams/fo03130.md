## 概要
府県週間予報は、天気、降水確率、気温、降水量を2日後から7日後までの1日毎の時系列で配信。

**資料名** <br/>
府県週間天気予報
 
**配信形式** <br/>
気象庁防災情報XMLフォーマット

**データ種類コード** <br/>
VPFW50

**/Report/Control/Title** <br/>
府県週間天気予報
 
**WebSocket圧縮形式** <br/>
gzip
