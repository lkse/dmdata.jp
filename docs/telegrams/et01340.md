## 概要
地震の活動状況等に関する情報や、伊豆東部の地震活動に関する情報などの解説情報を発表

**資料名** <br/>
 地震の活動状況等に関する情報
 
**配信形式** <br/>
 気象庁防災情報XMLフォーマット

**データ種類コード** <br/>
 VXSE56
 
**/Report/Control/Title** <br/>
 地震の活動状況等に関する情報

**WebSocket圧縮形式** <br/>
 gzip
