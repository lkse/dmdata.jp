## 概要

地上24時間予想図の高低気圧や前線等の各種天気図要素を緯度経度で示したもので、地図、前線記号、フォント等の情報は含まない。

**資料名** <br/>
地上24時間予想図（天気図情報）

**配信形式** <br/>
気象庁防災情報XMLフォーマット

**データ種類コード** <br/>
VZSF50

**/Report/Control/Title** <br/>
地上２４時間予想図

**WebSocket圧縮形式** <br/>
gzip
