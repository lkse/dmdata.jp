## 概要
日本周辺海域を航行する船舶等へ地方海上予報を発表。

**資料名** <br/>
 地方海上予報（Ｈ２８）
 
**配信形式** <br/>
 気象庁防災情報XMLフォーマット

**データ種類コード** <br/>
 VPCY51

**/Report/Control/Title** <br/>
 地方海上予報（Ｈ２８）
 
**WebSocket圧縮形式** <br/>
 gzip
