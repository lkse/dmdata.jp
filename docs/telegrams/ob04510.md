## 概要
気象台で観測した初雪、初氷、初霜、初冠雪を配信。

**資料名** <br/>
特殊気象報
 
**配信形式** <br/>
気象庁防災情報XMLフォーマット

**データ種類コード** <br/>
VGSK50

**/Report/Control/Title** <br/>
季節観測
 
**WebSocket圧縮形式** <br/>
なし

### 配信される電文の解説資料
