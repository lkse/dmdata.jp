---

## 概要
雷監視システム（LIDEN：LIghtning DEtection Network system）により雷を観測し、前1分間の観測結果を1分毎に配信する。

**資料名** <br/>
雷観測データ
 
**配信形式** <br/>
雷観測データ（VFJP40 RJTT）データフォーマット（資料参照）

なお、JMAソケットヘッダ、BCHについては、配信されるデータには付与されていません。
冒頭符号については、データ本文には含まれていません。

**データ種類コード** <br/>
VFJP40

**WebSocket圧縮形式** <br/>
なし
