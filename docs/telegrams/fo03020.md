## 概要
気圧配置の特性及び前線の状態、天気・気温・降水量、その他特記すべき異常気象等を府県予報区単位で発表。

**資料名** <br/>
天気概況
 
**配信形式** <br/>
気象庁防災情報XMLフォーマット

**データ種類コード** <br/>
VPFG50

**/Report/Control/Title** <br/>
天気概況
 
**WebSocket圧縮形式** <br/>
gzip
