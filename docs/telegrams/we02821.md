## 概要

水位周知河川で基準を超えた水位に基づき、受け持ち区間の浸水想定地区に対して洪水警報・注意報等、今後の予測・流域雨量を発表（水位観測所毎に1電文）

**配信開始時期未定**

**資料名** <br/>
水位周知河川に関する情報

**配信形式** <br/>
気象庁防災情報XMLフォーマット

**データ種類コード** <br/>
VXSUii (ii=50-59)

**/Report/Control/Title** <br/>
水位周知河川に関する情報

**WebSocket圧縮形式** <br/>
gzip
