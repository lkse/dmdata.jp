## 概要
衛星画像から解析した台風の中心位置、移動方位、速度、強度等（SAREP報）

**資料名** <br/>
気象衛星資料解析気象報
 
**配信形式** <br/>
国際気象通報式・別冊 FM94 BUFR 二進形式汎用気象通報式

**データ種類コード** <br/>
IUCC10

**WebSocket圧縮形式** <br/>
なし
