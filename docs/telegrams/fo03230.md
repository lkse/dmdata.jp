## 概要
2週間先にかけての全国的な推移を概況文として提供。

**資料名** <br/>
全般2週間気温予報
 
**配信形式** <br/>
気象庁防災情報XMLフォーマット

**データ種類コード** <br/>
VPZK70

**/Report/Control/Title** <br/>
全般季節予報（２週間気温予報）
 
**WebSocket圧縮形式** <br/>
gzip
