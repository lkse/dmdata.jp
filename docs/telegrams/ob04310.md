title: 潮位実況報
---

## 概要
気象庁が全国に設置している潮位観測点で観測された潮位データを潮位実況報として配信。

**資料名** <br/>
潮位実況報

**配信形式** <br/>
BUFR4

**データ種類コード** <br/>
ISTC61 <br/>
ISTC62

**WebSocket圧縮形式** <br/>
なし
