## 概要
特別警報の発表・継続・解除のみに特化した情報を発表

**2028年以降に配信終了**

**資料名** <br/>
 特別警報報知
 
**配信形式** <br/>
 気象庁防災情報XMLフォーマット

**データ種類コード** <br/>
 VPNO50

**/Report/Control/Title** <br/>
 気象特別警報報知
 
**WebSocket圧縮形式** <br/>
 gzip
