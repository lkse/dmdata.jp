## 概要
北西太平洋域でM6.5以上の地震が発生した場合、北西太平洋域の各国が津波警報等の発表に資するための支援情報として発表するものを複製した国内向け配信

**資料名** <br/>
 国際津波関連情報（国内向け）
 
**配信形式** <br/>
 A/N

**データ種類コード** <br/>
 WEPA60

**WebSocket圧縮形式** <br/>
 圧縮なし
