## 概要
指定河川で基準を超えた水位・水量を通知・予測情報、流域雨量観測を発表

**資料名** <br/>
 指定河川洪水予報（国土交通省と共同で行う指定河川洪水予報）
 
**配信形式** <br/>
 気象庁防災情報XMLフォーマット

**データ種類コード** <br/>
 VXKOii (ii=50-69)

**/Report/Control/Title** <br/>
 指定河川洪水予報
 
**WebSocket圧縮形式** <br/>
 gzip
