---
title: 潮位観測報
---

## 概要
気象庁が全国に設置している潮位観測点で観測された潮位データを速報的な品質管理を行い、波浪等の周期が短い成分を除いた高潮の防災上の利用に適したデータ。

**資料名** <br/>
潮位観測報

**配信形式** <br/>
BUFR4

**データ種類コード** <br/>
ISTC81 <br/>
ISTC82

**WebSocket圧縮形式** <br/>
なし
