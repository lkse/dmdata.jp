## 概要
日本周辺海域を航行する船舶等へ地方海上警報を発表。

**資料名** <br/>
 地方海上警報（Ｈ２８）
 
**配信形式** <br/>
 気象庁防災情報XMLフォーマット

**データ種類コード** <br/>
 VPCU51

**/Report/Control/Title** <br/>
 地方海上警報（Ｈ２８）
 
**WebSocket圧縮形式** <br/>
 gzip
