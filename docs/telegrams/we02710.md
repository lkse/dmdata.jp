## 概要
土砂災害の発生の恐れがある又は、おそれが少なくなった場合に発表

**2026年出水期を目途に土砂災害のレベル4の発表・解除に関する場合のみ運用に変更、2028年以降に配信終了**

**資料名** <br/>
土砂災害警戒情報
 
**配信形式** <br/>
気象庁防災情報XMLフォーマット

**データ種類コード** <br/>
VXWW50

**/Report/Control/Title** <br/>
土砂災害警戒情報
 
**WebSocket圧縮形式** <br/>
gzip
