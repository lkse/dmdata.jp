## 概要
前1時間に記録的な大雨を観測した場合発表

**2028年以降に配信終了**

**資料名** <br/>
 記録的短時間大雨情報
 
**配信形式** <br/>
 気象庁防災情報XMLフォーマット

**データ種類コード** <br/>
 VPOA50

**/Report/Control/Title** <br/>
 記録的短時間大雨情報
 
**WebSocket圧縮形式** <br/>
 gzip
