## 概要
台風又は熱帯的圧の実況、予報、台風による雨量・風・波浪・高潮などの予想、上陸・通過の通知を発表

**資料名** <br/>
 全般台風情報(総合情報、上陸等情報)
 
**配信形式** <br/>
 気象庁防災情報XMLフォーマット

**データ種類コード** <br/>
 VPTI50

**/Report/Control/Title** <br/>
 全般台風情報
 
**WebSocket圧縮形式** <br/>
 gzip
