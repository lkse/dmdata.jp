## 概要
各観測点における地上気象の観測値（雲量、雲の状態、雲の高さ、気圧（現地、海面）、気温、露点温度、風向、風速、視程、現在天気、過去天気、気圧変化の型、
気圧変化の量、降水量、最高気温、最低気温、積雪の深さ、降雪の深さ及び大気現象（S@AA21については、地吹雪、ホワイトアウトを追加））（SYNOP報）。

**資料名** <br/>
地上気象実況報
 
**配信形式** <br/>
国際気象通報式第8版 FM12 SYNOP－地上実況気象通報式

**データ種類コード** <br/>
