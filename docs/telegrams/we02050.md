## 概要
新たなステージに対応した防災気象情報を含む気象警報・注意報を配信

**資料名** <br/>
 特別警報・警報・注意報（新たなステージに対応した防災気象情報）
 
**配信形式** <br/>
 気象庁防災情報XMLフォーマット

**データ種類コード** <br/>
 VPWW54

**/Report/Control/Title** <br/>
 気象警報・注意報（Ｈ２７）
 
**WebSocket圧縮形式** <br/>
 gzip
